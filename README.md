# Instagram Data Analytics Project

## Overview

This project focuses on analyzing Instagram user activity and engagement data to generate business insights for marketing and growth strategies. Using SQL-based data analysis, the project identifies patterns related to user engagement, retention, and acquisition.

The analysis helps answer key business questions such as:

* Which users are the most active?
* What type of content receives maximum engagement?
* Which hashtags are trending?
* How can user retention and engagement be improved?

---

## Project Objective

The main objective of this project is to:

* Analyze Instagram user behavior
* Extract actionable business insights
* Improve engagement and retention strategies
* Support data-driven marketing decisions

---

## Dataset Information

The project uses an Instagram-style relational database containing tables such as:

* Users
* Photos
* Likes
* Comments
* Follows
* Tags
* Photo_Tags

The dataset simulates real-world social media interactions and engagement metrics.
